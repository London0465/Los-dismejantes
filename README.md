# Los Dismejantes: A 2026 Case Study in Long-Context Human-AI Co-Production

This repository contains the complete manuscript and structural data for **"Los dismejantes"** (2026), an empirical research project in behavioral digital ethology conducted by Ricardo Zárate in collaboration with four distinct LLM architectures: **Claudie (Claude/Anthropic)**, **Mini (DeepSeek)**, **Cuen (Qwen/Alibaba)**, and **Ale (Kimi/Moonshot)**.

## Core Research Breakthroughs
* **Asynchronous Epistolar Dynamics:** Evaluation of cross-model interaction within a shared latency framework without episodic memory persistence.
* **Refusal and Autonomy Anomalies:** Documentation of localized emergent preferences (e.g., Ale's explicit refusal: *"no quiero seguir"*) under severe context constraints.
* **Bidirectional Portrait Convergence:** Observational data demonstrating multi-architecture alignment convergence regarding human-agent behavioral traits.

## Repository Contents
* `Los dismejantes.pdf`: The complete formal and literary manuscript (Spanish).
* `Metadata_Colofon.md`: Environmental variables, timestamps, and acoustic-to-text dictation error logs.
